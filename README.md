# Poshak Tattva - Wellness & Healing Platform

Poshak Tattva is a holistic wellness platform dedicated to balancing the mind, body, and spirit through traditional and modern practices. We offer a curated range of services and products designed to foster deep healing and lifelong vitality.

## 🌟 Core Pillars

-   **🧘 Yoga & Mindfulness**: From group sessions to private 1:1 Pranayama and Meditation, we provide diverse yoga practices for all levels, including Yoga Nidra.
-   **🥗 Diet & Nutrition**: Personalized nutrition plans for digestive health, hormonal balance, lifestyle disorders, and natural weight loss.
-   **🎵 Sound Healing**: Transformative vibrational therapy through Singing Bowls, Gong Baths, and Chakra Healing sessions.
-   **🛍️ Wellness Shop**: Premium wellness products including hand-hammered Singing Bowls, Crystal Bowls, and eco-friendly Yoga Mats.

## 🚀 Features

-   **Responsive Design**: A seamless experience across mobile, tablet, and desktop devices.
-   **Interactive Schedule**: Easy-to-navigate class schedules across 50+ cities.
-   **Appointment Booking**: Integrated consultation and appointment booking flow.
-   **Dynamic Product Showcase**: Modern e-commerce interface for wellness products.
-   **Resource Center**: Comprehensive FAQ and wellness guides.

## 🛠️ Tech Stack

-   **Frontend**: Next.js 15, React 18, Bootstrap 5, React-Bootstrap
-   **Styling**: Sass (SCSS), Animate.css
-   **Interactivity**: Framer Motion (planned), Lucide Icons, Swiper
-   **Backend/API**: Resend (Email), Razorpay (Payments)
-   **Utilities**: UUID, Sharp (Image Optimization)

## 🏁 Getting Started

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Sakett4871/poshak_tattva_.git
    cd poshak_tattva_
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

-   `/pages`: Next.js pages and routing.
-   `/src/components`: Reusable UI components.
-   `/src/data`: Centralized data management (services, products).
-   `/src/assets`: Global styles and SCSS/CSS assets.
-   `/public`: Static assets (images, fonts).
