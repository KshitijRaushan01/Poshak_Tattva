# Contributing to Poshak Tattva

Thank you for your interest in contributing to Poshak Tattva! We welcome contributions that help improve our wellness platform.

## 📜 Code of Conduct

Please be respectful and professional in all interactions.

## 🛠️ How to Contribute

1.  **Fork the Repository**: Create your own fork of the project.
2.  **Create a Feature Branch**: `git checkout -b feature/your-feature-name`.
3.  **Make Changes**: Ensure your code follows the existing style and is well-documented.
4.  **Test Your Changes**: Run `npm run dev` and verify your changes locally.
5.  **Commit Your Changes**: Use descriptive commit messages.
6.  **Push and Pull Request**: Push to your fork and submit a pull request to the main repository.

## 🎨 Style Guide

-   Use functional components and React hooks.
-   Follow Bootstrap 5 conventions for layout and responsiveness.
-   Use SCSS for custom styling, keeping it modular.
-   Maintain clear and descriptive naming conventions for components and variables.

## 🐛 Reporting Bugs

If you find a bug, please create an issue with details on how to reproduce it and your environment.
